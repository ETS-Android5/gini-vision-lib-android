GiniVisionSpecificExtraction(@NonNull name: String, @NonNull value: String, @NonNull entity: String, @Nullable box: GiniVisionBox?, @NonNull candidates: MutableList<GiniVisionExtraction!>)

Value object for a specific extraction from the Gini API.

Parameters

name - String: The specific extraction's name, e.g. "amountToPay". See Available Specific Extractions for a full list

value - String: normalized textual representation of the text/information provided by the extraction value (e. g. bank number without spaces between the digits). Changing this value marks the extraction as dirty

entity - String: key (primary identification) of an entity type (e.g. banknumber). See Extraction Entities for a full list

box - GiniVisionBox?: (optional) bounding box containing the position of the extraction value on the document. Only available for some extractions. Changing this value marks the extraction as dirty

candidates - MutableList<GiniVisionExtraction!>: A list containing other candidates for this specific extraction. Candidates are of the same entity as the found extraction
