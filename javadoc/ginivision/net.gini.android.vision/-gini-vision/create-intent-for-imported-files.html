createIntentForImportedFiles

@NonNull open fun createIntentForImportedFiles(@NonNull intent: Intent, @NonNull context: Context, @NonNull callback: AsyncCallback<Intent!, ImportedFileValidationException!>): CancellationToken

Screen API

If you have enabled the multi-page feature and your application receives one or multiple files from another application you can use this method to create an Intent for launching the Gini Vision Library.

Importing the files is executed on a secondary thread as it can take several seconds for the process to complete. The callback methods are invoked on the main thread.

In your callback's onSuccess(Intent) method start the Intent with to receive the extractions or a GiniVisionError in case there was an error.

Parameters

intent - Intent: the Intent your app received

context - Context: Android context

callback - AsyncCallback<Intent!, ImportedFileValidationException!>: A AsyncCallback implementation

Return
CancellationToken: a CancellationToken for cancelling the import process
