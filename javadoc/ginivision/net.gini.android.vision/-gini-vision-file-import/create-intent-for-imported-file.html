createIntentForImportedFile

@NonNull static fun createIntentForImportedFile(@NonNull intent: Intent, @NonNull context: Context, @NonNull reviewActivityClass: Class<out ReviewActivity!>, @NonNull analysisActivityClass: Class<out AnalysisActivity!>): Intent
Deprecated: Deprecated in Java

Screen API

When your application receives a file from another application you can use this method to create an Intent for launching the Gini Vision Library.

Start the Intent with android.app.Activity#startActivityForResult(Intent, int) to receive the GiniVisionError in case there was an error.

Parameters

intent - Intent: the Intent your app received

context - Context: Android context

reviewActivityClass - Class<out ReviewActivity!>: the class of your application's ReviewActivity subclass

analysisActivityClass - Class<out AnalysisActivity!>: the class of your application's AnalysisActivity subclass

Exceptions

ImportedFileValidationException - if the file didn't pass validation

IllegalArgumentException - if the Intent's data is not valid or the mime type is not supported

Return
Intent: an Intent for launching the Gini Vision Library
