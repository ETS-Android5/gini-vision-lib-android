GiniVisionDebug

class GiniVisionDebug

This class allows you to enable and disable debugging for the Gini Vision Library.

Debugging is disabled by default.

Warning: Don't forget to disable debugging before releasing.

If debug is enabled:

  • The reviewed jpegs are written to a folder called ginivisionlib in your app's external directory.
