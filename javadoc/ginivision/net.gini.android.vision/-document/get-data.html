getData

@Nullable abstract fun getData(): ByteArray?

The contents of a document, if the document was loaded into memory.

For photos captured with the camera or for QR Codes this is never null.

If Document#isImported() is true then this might be null. If it's null you can use Document#getIntent() and access the contents using the Intent.

Return
ByteArray?: a byte array containing one of the supported document types
