EXTRA_IN_SHOW_ONBOARDING

static val EXTRA_IN_SHOW_ONBOARDING: String

Optional extra which must contain a boolean and indicates whether the Onboarding Screen should be shown when the Gini Vision Library is started.

Default value is false.

Deprecated
Configuration should be applied by creating a GiniVision instance using GiniVision#newInstance() and the returned GiniVision.Builder.
