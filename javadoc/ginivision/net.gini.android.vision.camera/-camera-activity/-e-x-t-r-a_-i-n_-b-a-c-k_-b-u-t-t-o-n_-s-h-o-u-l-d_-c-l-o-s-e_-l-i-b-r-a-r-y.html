EXTRA_IN_BACK_BUTTON_SHOULD_CLOSE_LIBRARY

static val EXTRA_IN_BACK_BUTTON_SHOULD_CLOSE_LIBRARY: String

Optional extra wich must contain a boolean and indicates whether the back button should close the Gini Vision Library.

Default value is false.

Deprecated
The option to close the library with the back button from any screen will be removed in a future version.
