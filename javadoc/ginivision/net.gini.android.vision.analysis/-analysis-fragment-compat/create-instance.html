createInstance

open static fun createInstance(@NonNull document: Document, @Nullable documentAnalysisErrorMessage: String?): AnalysisFragmentCompat!

Factory method for creating a new instance of the Fragment using the provided document.

You may pass in an optional analysis error message. This error message is shown to the user with a retry button.

Note: Always use this method to create new instances. Document is required and an exception is thrown if it's missing.

Parameters

document - Document: must be the Document from ReviewFragmentListener#onProceedToAnalysisScreen

documentAnalysisErrorMessage - String?: an optional error message shown to the user

Return
AnalysisFragmentCompat!: a new instance of the Fragment
