open fun onAddDataToResult(result: Intent!): Unit
Deprecated: Deprecated in Java

Callback for adding your own data to the Activity's result.

Called when the document was analyzed.

You should add the results of the analysis as extras and retrieve them when the returned.

Note: you must call AnalysisActivity#onDocumentAnalyzed() after you received the analysis results from the Gini API, otherwise this method won't be invoked.

Parameters

result - Intent!: the Intent which will be returned as the result data.

Deprecated
When a GiniVision instance is available the document is analyzed internally by using the configured implementation. The extractions will be returned in the extra called of the CameraActivity's result Intent.
