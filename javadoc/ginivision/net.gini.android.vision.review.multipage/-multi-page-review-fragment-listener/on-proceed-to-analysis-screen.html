onProceedToAnalysisScreen

abstract fun onProceedToAnalysisScreen(@NonNull document: GiniVisionMultiPageDocument<GiniVisionDocument!, GiniVisionDocumentError!>): Unit

Called when all pages were uploaded successfully and the user tapped on the "next" button.

If you use the Screen API you should start the AnalysisActivity and set the document as the AnalysisActivity#EXTRA_IN_DOCUMENT extra.

If you use the Component API you should start the AnalysisFragmentCompat (or the AnalysisFragmentStandard) and pass the document when creating it with (or ).

Parameters

document - GiniVisionMultiPageDocument<GiniVisionDocument!, GiniVisionDocumentError!>: contains the reviewed image (can be the original one or a modified image)
