abstract fun onProceedToAnalysisScreen(@NonNull document: Document, @Nullable errorMessage: String?): Unit

Called when the user tapped on the Next button and one of the following conditions apply:

  • Analysis is in progress
  • Analysis completed with an error
  • The image was rotated

    You should start your Activity extending AnalysisActivity and set the document as the AnalysisActivity#EXTRA_IN_DOCUMENT extra.

Parameters

document - Document: contains the reviewed image (can be the original one or a modified image)

errorMessage - String?: an optional error message to be passed to the Analysis Screen
