onDocumentWasRotated

abstract fun onDocumentWasRotated(@NonNull document: Document, oldRotation: Int, newRotation: Int): Unit
Deprecated: Deprecated in Java

Called when the user rotated the image.

In case you started the document analysis in ReviewFragmentListener#onShouldAnalyzeDocument(Document) you should cancel it here as the original image is not valid anymore.

Parameters

document - Document: contains the modified image

oldRotation - Int: the previous rotation in degrees

newRotation - Int: the new rotation in degrees

Deprecated
When a GiniVision and a GiniVisionNetworkService instance is available rotation is handled internally. The document is analyzed by using the configured GiniVisionNetworkService implementation. The extractions will be returned in the Analysis Screen in AnalysisFragmentListener#onExtractionsAvailable(Map, Map).
