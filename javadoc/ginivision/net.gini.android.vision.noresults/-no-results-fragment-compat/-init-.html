Component API

When you use the Component API with the Android Support Library, the NoResultsFragmentCompat displays hints that show how to best take a picture of a document.

Include the NoResultsFragmentCompat into your layout by using the factory method to create an instance and display it using the androidx.fragment.app.FragmentManager.

Your Activity must implement the NoResultsFragmentListener interface to receive events from the No Results Fragment. Failing to do so will throw an exception.

Your Activity is automatically set as the listener in NoResultsFragmentCompat#onCreate(Bundle).
