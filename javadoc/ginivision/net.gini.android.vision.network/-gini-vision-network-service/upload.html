abstract fun upload(@NonNull document: Document, @NonNull callback: GiniVisionNetworkCallback<Result!, Error!>): CancellationToken!

Called when a document needs to be uploaded to the Gini API.

You should only upload the document. Polling or retrieving extractions is not needed at this point.

Parameters

callback - GiniVisionNetworkCallback<Result!, Error!>: a callback implementation to return the outcome of the upload

Return
