Package net.gini.android.vision.tracking

Contains the interface, enums and classes used for tracking various events which occur during the usage of the Gini Vision Library.

Implement the net.gini.android.vision.tracking.EventTracker interface and pass it to the net.gini.android.vision.GiniVision.Builder.setEventTrackernet.gini.android.vision.tracking.EventTracker when creating a new GiniVision instance.
